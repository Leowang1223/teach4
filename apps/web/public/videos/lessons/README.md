# 課程影片目錄

## 📁 目錄結構

```
videos/
└── lessons/
    ├── L1/
    │   ├── step1.mp4  (你好 - Hello)
    │   ├── step2.mp4  (我是學生 - I am a student)
    │   ├── step3.mp4  (造句練習 - Sentence practice)
    │   └── step4.mp4  (綜合複習 - Review)
    ├── L2/
    │   └── (待新增)
    └── L3/
        └── (待新增)
```

## 📝 使用說明

### 1️⃣ 放置影片檔案
請將對應的 MP4 影片檔案放入各課程目錄中。例如：
- `L1/step1.mp4` - Lesson 1 第一步影片
- `L1/step2.mp4` - Lesson 1 第二步影片
- 以此類推...

### 2️⃣ 影片規格建議
- **格式**: MP4 (H.264 編碼)
- **解析度**: 1080p 或 720p
- **長寬比**: 16:9 或 9:16 (根據內容調整)
- **檔案大小**: 建議每個影片 < 50MB

### 3️⃣ 課程資料設定
在 `apps/backend/src/plugins/chinese-lessons/L1.json` 中：
```json
{
  "id": 1,
  "video_url": "/videos/lessons/L1/step1.mp4"
}
```

### 4️⃣ 前端顯示設定
影片會使用以下樣式顯示：
- 高度: 66vh (佔畫面 2/3)
- 寬度: 100%
- 圓角: 16px
- 自適應: `object-fit: cover` (避免黑邊)
- 控制器: 原生控制器，但限制下載、全螢幕、遠端播放

## ⚠️ 注意事項

1. **路徑規則**: 所有影片路徑都從 `/videos/` 開始
2. **檔案命名**: 使用 `stepX.mp4` 格式，X 為步驟編號
3. **版權**: 確保所有影片都有適當的使用權限
4. **效能**: 大型影片檔案可能影響載入速度，建議壓縮後再使用

## 🔄 從 YouTube 遷移

如果你之前使用 YouTube 連結，現在需要：
1. 下載 YouTube 影片 (確保有權限)
2. 轉換為 MP4 格式
3. 放入對應的課程目錄
4. 更新 JSON 檔案中的 `video_url`
